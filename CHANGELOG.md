# Changelog

All notable changes to this project are documented here. The format follows
[Keep a Changelog](https://keepachangelog.com/en/1.1.0/), and the project aims to
follow [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

GitHub release notes (generated by GoReleaser from commit history) accompany each
tagged release; this file is the human-curated summary.

## [Unreleased]

### Added

- MIT `LICENSE`.
- Pinned developer toolchain via [mise](https://mise.jdx.dev) (Go,
  golangci-lint, goreleaser, govulncheck) and a [Task](https://taskfile.dev)
  `Taskfile.yml` with a single `task ci` gate (fmt-check, vet, lint, test, vuln).
- Continuous integration on Linux and macOS, and a tag-driven release pipeline
  (GoReleaser) that publishes cross-platform binaries.
- Installation instructions (`go install` and release binaries) and a "how it
  works" flow guide.
- Community-health files: security policy, code of conduct, issue and pull
  request templates, `CODEOWNERS`, and `.editorconfig`.

### Changed

- Hardened the contract linker and SQLite store against path traversal and
  unchecked close errors.

### Security

- Pinned Go to 1.26.4 to clear two reachable standard-library advisories
  (`GO-2026-5039` in `net/textproto`, `GO-2026-5037` in `crypto/x509`).
