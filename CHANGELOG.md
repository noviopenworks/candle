## [1.0.0] - 2026-07-08

First public release. candle joins a Graphify code graph, OpenAPI/protobuf
contracts, and Go private-library usage into one MCP-served, SQLite-backed
knowledge graph — linked back to code so an agent can answer "which handler
implements this endpoint?" and "what breaks if I change this?" across repos.

### Added
- **16 MCP tools over stdio** — code graph (`list_repos`, `resolve_repo`,
  `get_context`, `query_repo`, `explain_symbol`, `get_file_context`,
  `call_path`), OpenAPI (`list_apis`, `find_endpoint`, `explain_endpoint`,
  `find_schema`), protobuf (`find_rpc`, `explain_rpc`), and private libraries
  (`find_private_library`, `find_library_consumers`, `explain_private_library`).
- **5 commit-pinned resource URI schemes**: `repo://`, `graph://`, `openapi://`,
  `proto://`, `lib://`.
- **Contract → code linking**: OpenAPI operations and proto RPCs linked to
  implementing symbols (AST-confirmed HIGH tier when a source `root` is set,
  heuristic MEDIUM/LOW otherwise).
- **`call_path`**: multi-hop call traversal (up to 5 hops, callees/callers/both)
  returned as a tree, with per-path cycle cutting.
- **Cross-repo aggregation**: RPC `consumed_by` (heuristic: repos with a node
  labelled like the RPC, excluding providers) and private-library consumers
  across indexed repos.
- **Observability**: `--verbose` structured logging on stderr (stderr-safe for
  the stdio transport); not-found results carry a reason on every tool.
- **Stability policy**: per-tool/resource stable/experimental tiers and a
  semver/deprecation policy ([`docs/stability.md`](docs/stability.md)).
- **Graphify quickstart**: the verified `graph.json` contract
  ([`docs/graphify-quickstart.md`](docs/graphify-quickstart.md)).
- Idempotent per-repo indexing, scoped multi-instance serving, and MCP client
  configs for opencode and Claude Desktop/Code ([`examples/clients/`](examples/)).
- Installation instructions (`go install` and release binaries) and a
  how-it-works [flow guide](flow.md).
- MIT `LICENSE`; pinned toolchain via [mise](https://mise.jdx.dev) and a
  `task ci` gate (fmt-check, vet, lint, test, vuln); CI on Linux and macOS;
  tag-driven GoReleaser pipeline publishing cross-platform binaries.
- Community-health files: security policy, code of conduct, issue/PR templates,
  `CODEOWNERS`, `.editorconfig`.

### Changed
- Default config file renamed to `candle.yaml` (was `manifest.yaml`).
- `explain_rpc`'s `consumed_by` widened from a placeholder string to a `[]string`
  of consumer repos.
- Hardened the contract linker and SQLite store against path traversal and
  unchecked close errors.

### Security
- Pinned Go to 1.26.5 to clear reachable standard-library advisories
  (`GO-2026-5039` in `net/textproto`, `GO-2026-5037` in `crypto/x509`,
  `GO-2026-5856` in `crypto/tls`).
